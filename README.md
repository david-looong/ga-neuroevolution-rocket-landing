# Comparing the Efficiency of Evolutionary Algorithms and Particle Swarm Optimization in 2D Rocket Landing

**Team Members:** Tyler Garriott, David Long, Gavin Picard

This project serves as our final project for our COSC420/527 course, Introduction to Biologically-Inspired Computing, at The University of Tennessee, Knoxville.

**RQ1:** How does an evolutionary algorithm compare to particle swarm optimization in learning control polocies for a 2D rocket landing task?

To investigate this, we will implement a 2D physics simulation with simplified dynamics, including gravity and drag. We will represent control policies as sequences of thrust magnitudes and angles over time, then apply both an evolutionary algorithm and particle swarm optimization to learn these policies. We plan to. evaluate performance using metrics such as landing success rate, landing stability, and convergnce speed.
